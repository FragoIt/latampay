# Instrucciones para la Demo

¡Todo listo para tu presentación!

## 1. Cargar Datos
Ejecuta el contenido de `supabase/seed_demo.sql` en el SQL Editor de tu proyecto en Supabase.
Esto borrará los datos anteriores y cargará eventos realistas.

## 2. Iniciar
```bash
npm run dev
```
Abre http://localhost:3000

## 3. Guión
Sigue los pasos en `demo_script.md` (en la carpeta de artifacts o en el "Brain").
Recuerda mostrar:
1. Dashboard KPIs (Equipos Afuera).
2. Inventario Visual.
3. Crear Renta (3 clics).
4. **NUEVO**: Descargar PDF de contrato.
5. **NUEVO**: Enviar WhatsApp.

¡Éxito!
